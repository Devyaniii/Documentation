Interface IDrawStrategy

Namespace
ASE_Application_Demo
Assembly
ASE Application Demo.dll

Defines the methods for drawing basic shapes and text on a graphics canvas. Implementations of this interface should provide the specific drawing behavior for different strategies (e.g., solid, dashed).

public interface IDrawStrategy

DrawCircle(Graphics, Pen, int, int, int, bool)

Draws a circle on the canvas at the specified position and radius. The circle can be filled or outlined based on the filled parameter.
