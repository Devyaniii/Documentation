Namespace ASE_Application_Demo

Classes

AppCanvas

Represents a drawable canvas for creating graphical shapes, text, and images.

AppCommandFactory

Factory class responsible for creating application-specific commands.

AppStoredProgram

Represents a stored program that manages the execution of commands on a given canvas. Inherits from the BOOSE.StoredProgram class and overrides the Run() method to execute a series of commands until none are left to execute.

DashedDrawStrategy

Provides drawing operations for basic shapes and text using a dashed line strategy. Implements the IDrawStrategy interface to define how shapes and text are drawn on the canvas with dashed lines.

Form1

Represents the main form of the application, handling user interactions, program parsing, error handling, and canvas rendering.

SolidDrawStrategy

Provides drawing operations for basic shapes and text using a solid fill strategy. Implements the IDrawStrategy interface to define how shapes and text are drawn on the canvas.

Interfaces

IDrawStrategy

Defines the methods for drawing basic shapes and text on a graphics canvas. Implementations of this interface should provide the specific drawing behavior for different strategies (e.g., solid, dashed).
